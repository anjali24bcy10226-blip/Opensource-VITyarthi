# Linux Audit Project – Anjali Patni

## 📌 Project Overview
This project explores Linux and open-source philosophy through:
1. A structured audit of an open-source software project
2. Five shell scripts demonstrating practical Linux skills

The scripts reflect core Linux concepts such as automation, transparency, and system interaction.

---

## 🛠️ Scripts Included

### Script 1 — System Identity Report
Displays system information like OS, user, uptime, and license.

### Script 2 — FOSS Package Inspector
Checks if a package is installed, shows its version, and describes its purpose.

### Script 3 — Disk and Permission Auditor
Analyzes disk usage, ownership, and permissions of key directories.

### Script 4 — Log File Analyzer
Reads a log file and counts occurrences of keywords like ERROR or WARNING.

### Script 5 — Open Source Manifesto Generator
Generates a personalized open-source philosophy statement.

---

## 💡 Concepts Covered
- Variables and command substitution
- Conditional statements (if-then, case)
- Loops (for, while)
- File handling and text processing
- User interaction and automation
